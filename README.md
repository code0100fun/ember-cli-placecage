# ember-cli-placecage

[![PlaceCage](http://www.placecage.com/gif/400/300)](http://www.placecage.com/)

## Usage

```html
<place-cage class="avatar" alt={{user.name}} width="80" height="90"/>
```

## Development

* `git clone` this repository
* `npm install`
* `bower install`

## Running

* `ember server`
* Visit your app at http://localhost:4200.

## Running Tests

* `ember test`
* `ember test --server`

## Building

* `ember build`
